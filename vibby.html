Connecter Stripe
🔒 Vibby ne demande jamais ta clé secrète Stripe. On a seulement besoin de ta clé publique (commence par pk_) et d'un lien de paiement par produit — les deux sont sûrs à coller ici, rien n'est envoyé à un serveur, tout reste dans ton navigateur.
Trouve-la dans ton dashboard Stripe → Développeurs → Clés API.
💡 Pour chaque produit ci-dessus, crée un Payment Link dans Stripe (aucun code requis, ~1 min) puis colle l'URL dans le champ "Lien de paiement Stripe". Créer un Payment Link → Active "quantité modifiable" si tu veux permettre plusieurs unités.

Héberge ce fichier n'importe où (Netlify, Vercel, GitHub Pages, ton propre serveur) — il n'a besoin d'aucun backend pour fonctionner. Le suivi des ventes/visites, lui, a besoin d'internet pour parler à ton tableau de bord Vibby (déjà branché).

Pour un panier unifié en un seul paiement, déploie ce petit serveur — ta clé secrète Stripe reste chez toi, jamais partagée avec Vibby.
Déploie-le en 5 min sur Vercel/Render avec ta clé STRIPE_SECRET_KEY en variable d'environnement, puis colle l'URL de ton endpoint dans le fichier généré (constante VIBBY_CHECKOUT_ENDPOINT).
